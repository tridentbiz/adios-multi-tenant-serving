## Installation

### Standalone Installation
```bash
git clone https://github.com/tridentbiz/adios-multi-tenant-serving.git
cd adios-multi-tenant-serving
./install.sh
```

### AdiOS Integration
```bash
cd /path/to/0001-adios
git submodule add https://github.com/tridentbiz/adios-multi-tenant-serving.git plugins/multi-tenant-serving
adios plugin install multi-tenant-serving
```
